# TextRank4ZH

TextRank算法可以用来从文本中提取关键词和摘要。
## 安装

方式1：

$ python setup.py install --user
```


方式2：
```
$ sudo python setup.py install
```


方式3：
```
$ pip install textrank4zh --user
```


方式4：
```
$ sudo pip install textrank4zh
